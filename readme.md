# "Join Us"

A game made for the [2026 Ebitengine Game Jam](https://itch.io/jam/ebitengine-game-jam-2026)

You live in a tech dystopia where a robot hive is taking over the world. Your aim is to break into a facility creating mind-controlling technology.

Take over the facility from the inside to save the planet.
